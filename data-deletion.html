📦 એપ કયા પ્રકારનો ડેટા સંગ્રહ કરે છે?

📱 Local Storage

TA entries, duty records, generated PDF history

☁️ Firebase Auth

Registered mobile number, login session, registration form data

🔄 ડેટા ક્યારે ડીલીટ થાય?

🗑️
App Uninstall કરો Automatic
Local storage નો બધો TA data તરત જ delete થઈ જાય છે — TA entries, Duty records, PDF history — કોઈ manual request જરૂરી નથી.
📲
Firebase Account Data Request જરૂરી
Firebase Authentication માં store mobile number અને registration data delete કરવા Help & Support નો સંપર્ક કરવો પડે.
🔁
ફરી Login કરો
App reinstall કર્યા પછી same number થી login કરી શકો છો. TA entries fresh start — નવેસરથી entry કરી શકો છો.

✅ App uninstall = TA data automatic delete — કોઈ request વગર.
✅ Firebase account delete = Support contact કરો — 30 days માં remove.
✅ Re-registration = ગમે ત્યારે same number થી ફરી account બનાવી શકો.

📌 નોંધ: Prasthan app account delete કર્યા વિના કારણ કે TA entries device ના local storage માં save છે — app delete = data delete.
